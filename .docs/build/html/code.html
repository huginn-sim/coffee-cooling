coffee.cool(times, t0, dt, T0, Tf, Tp, Te)[source]¶

Calculates a set of Temperature values given some initial conditions and calculated constants.

Parameters:
  • times – A dictionary of time measurements. (‘black’:interval, ‘cream’:samples of ‘black’)
  • t0 – The inital time of Temperature measurement.
  • dt – The time differential.
  • T0 – The initial temperature of the object.
  • Tf – The temperature equilibrium.
  • Tp – The preferred temperature of the object.
  • Te – The immediate change in temperature upon experiment.
coffee.model(t0=0, tf=30, dt=0.1, T0=90, Tf=70, Tp=75, Te=-5, experiments=3)[source]¶

Models the rate of cooling of coffee over time.

Parameters:
  • t0 – The inital time of Temperature measurement.
  • tf – The maximum amount of time allotted for cooling.
  • dt – The time differential.
  • T0 – The initial temperature of the object.
  • Tf – The temperature equilibrium.
  • Tp – The preferred temperature of the object.
  • Te – The immediate change in temperature upon experiment.
  • experiments – The number of intervals to conduct an experiment; add cream.
coffee.sample_data(data=None)[source]¶

Calculates thermal constants from a dataset.

Parameters:data – The dataset used to derive cooling constants. If ‘None’ use default ‘data’.
